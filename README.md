# PDF Text Extractor (OCR) 📄🔍

A **Streamlit** app that extracts text from scanned PDFs using **Tesseract OCR**.

## Features
✅ Upload a **PDF file**  
✅ Convert scanned pages to **text**  
✅ **Download** the extracted text
